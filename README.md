# MEDICON – ICU Resource Allocation Under Crisis

## Overview
MEDICON is an AI-based decision support system designed to optimize ICU resource allocation during crisis situations such as pandemics or natural disasters.

The system analyzes patient data and hospital resources to allocate ICU beds efficiently while maintaining fairness and ethical decision-making.

---

## Problem Statement
During crisis situations:
- ICU beds and ventilators are limited  
- Patient inflow is dynamic  
- Manual decision-making is slow and inconsistent  
- Ethical conflicts arise between fairness and survival  

This project aims to solve these challenges using data-driven techniques.

---

## Objectives
- Allocate ICU beds efficiently  
- Predict patient survival probability using Machine Learning  
- Classify patients based on severity  
- Apply ethical decision-making rules  
- Provide a real-time dashboard for monitoring  

---

## Technologies Used
- Python  
- Pandas, NumPy  
- Scikit-learn  
- Plotly  
- Streamlit  
- Folium  

---

## Methodology
1. Data collection (patient and hospital data)  
2. Data preprocessing and cleaning  
3. Severity classification using clustering  
4. Survival prediction using Logistic Regression  
5. Ethical rule-based allocation  
6. Visualization using Streamlit dashboard  

---

## Features
- Patient prioritization based on severity  
- Survival probability prediction  
- ICU allocation recommendations  
- Hospital resource tracking  
- Interactive dashboard  
- Geographic visualization of patients and hospitals  

---

## Output
- Dashboard showing:
  - Patient records  
  - ICU allocation status  
  - Resource availability  
  - Allocation recommendations
